1. Install dependencies

```bash
npm install
```

2. Run dev server (Vite via Astro)

```bash
npm run dev
```

3. Build

```bash
npm run build
```

Notes:

- SCSS is compiled via `sass` and handled by Astro/Vite automatically.
- Posts should live in `src/content/posts/` (Markdown files) for the best authoring experience. If you prefer to keep `starter-code/data.json`, `src/data/posts.ts` will import and normalize that file.
